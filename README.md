# Purple Bags

Welcome to the **Purple Bags** project! This is a modern, responsive web application built with [Next.js](https://nextjs.org), designed to showcase a premium collection of bags with an immersive user experience.

## ✨ Features

- **Dynamic Hero Section:** A custom, math-driven animated 180-degree semicircle (arc) showcasing featured bags right on the landing page, built with Framer Motion.
- **Modern UI:** Built using Tailwind CSS for a fully responsive, sleek, and premium design.
- **Interactive Animations:** Smooth hover effects and entry animations using Framer Motion to wow the user.
- **Optimized Assets:** High-performance image loading using Next.js `next/image`.

## 🚀 Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app in action.

## 📁 Project Structure

- `src/app/page.tsx`: The main landing page.
- `src/components/ui/HeroBags.tsx`: The custom animated component that powers the 180-degree interactive bag arc.
